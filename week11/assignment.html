Export a Glitch site to GitHub Pages

Export one of your Glitch sites to GitHub Pages:

  1. Create a repository on GitHub Pages,
  2. export your site to that repository,
  3. create and merge a pull request to put your Glitch code on the master branch,
  4. turn on GitHub Pages for your repository on the master branch,
  5. ensure that the site is visible on GitHub Pages--the URL should start with yourusername.github.io. For example, https://brelsfoe.github.io/tiger-sharks/
